## MadeMeSmile

An app to build a chain of good memories.

Every time you smile, you’ll take a selfie with a good note

### MVP features

- **Add a Moment**: User types a short note, uploads a picture/video, or records an audio clip.
- **Timeline/List View**: Display moments in a chronological feed.
- **Simple Categories/Tags**: Like _Family, Friends, Work, Random_ for light organization.
- **Daily Reminder**: Push notification encouraging the user to “add one thing that made you smile today.”
